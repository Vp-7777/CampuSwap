Option 2: Fresh Maven Install
------------------------------
1. Download Maven from: https://maven.apache.org/download.cgi
2. Extract the zip file (NOT to Downloads, use C:\Program Files or C:\Tools)
3. Add to PATH: [extract location]\apache-maven-3.9.11\bin
4. Verify: Open new CMD and type: mvn --version

HOW TO UPDATE PATH:
===================
1. Press Windows Key
2. Type "Environment Variables"
3. Click "Edit system environment variables"
4. Click "Environment Variables" button
5. Under "System variables", find "Path"
6. Click "Edit"
7. Fix the Maven path or add correct one
8. Click OK on all windows
9. RESTART any open terminals
